# Mini SOC Lab – Splunk Based Detection Lab

## Overview
This project simulates a small SOC environment using:
- Kali Linux (attacker)
- Ubuntu Server (victim)
- Splunk (SIEM)

## Scenarios Implemented
- SSH Brute Force Attack
- Detection via Splunk queries
- Alert creation
- Dashboard creation
- Privilege escalation activity monitoring
- Firewall containment using UFW

## MITRE ATT&CK Techniques
- T1110 – Brute Force
- T1078 – Valid Accounts
- T1068 – Privilege Escalation
- T1562 – Impair Defenses
